## 分类规则

- 能独立运行、独立展示、独立部署的内容，优先独立建仓库。
- 同类型的小作品，可以先放在合集仓库，成熟后再拆分。
- 参考案例和模板不要混到正式项目里。
- 仓库名尽量使用英文，README 可以使用中文。
- GitHub Topics 使用英文，方便搜索、筛选和后续自动归类。
